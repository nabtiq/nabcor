# Rubric — Brand Fidelity (Mode B)

**Used by:** evaluate-brand-fidelity. **Authority:** ADVISORY.
Scores channel specs/renders against the verified `brand-dna` (never against taste).

| Verdict | Anchor |
|---|---|
| `faithful` | verified identity elements (names, marks, palette where verified, voice never-list) all honored; deviations are recorded in visual-world.deviations_from_dna with approval |
| `deviating-approved` | deviations exist, all recorded + approved (this is fine — it's the BC-001 display-gold pattern: propose, flag, get sign-off) |
| `deviating-unapproved` | any identity deviation without a recorded approval → finding per deviation |
| `violating` | contradicts a verified identity fact or a strict-guideline constraint |

**Evidence required:** per-element check table (names / marks / palette / type /
voice / identity_constraints) with claim refs. Unverified (`proposed`) identity
elements are checked for *internal consistency*, not fidelity — noted as such.
