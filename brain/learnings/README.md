# Learnings — append-only memory

Learnings are reusable behavior changes backed by evidence. They are never silently
edited or deleted; a correction appends a new record with `supersedes`.

Use `brain/templates/LEARNING_TEMPLATE.md`. Brand-specific learnings must remain inside
their brand namespace when tenancy exists. Cross-brand aggregation is limited to
rights-clean, non-content patterns.
