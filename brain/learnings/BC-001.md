## Supported lessons retained

- Typed, durable artifacts reduce drift between work rounds.
- Human direction and acceptance gates carry disproportionate quality value.
- Deterministic validation should govern generative output.
- Arabic, theme, viewport, motion, and JavaScript-disabled states need explicit
  validation declarations when rendered channels exist.
- Claims and asset rights must be tracked before production use.
- Token, cache, run, context, and artifact-lifecycle records must exist by construction;
  reconstructing them later is unreliable.
- A single orchestrator with bounded skills is the default; more agents require a
  measured bottleneck.
- Working memory is ephemeral; canonical artifacts and commit checkpoints are durable.

## Evidence boundary

BC-001 was one bilingual construction-sector website project with a strong operator.
It does not prove universal token savings, cross-channel demand, genericity detection,
or the exact budgets proposed for the new slice. Those remain experiments.
